## Getting Started

### Prerequisites

* node.js
* npm

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/lopezrunco/whatsapp-automation-system-frontend.git
   ```
3. Install NPM packages
   ```sh
   npm i
   ```
4. Run app
   ```sh
   npm start
   ```
